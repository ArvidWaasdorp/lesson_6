With jQuery you select (query) HTML elements and perform "actions" on them. The jQuery syntax is tailor-made for selecting HTML elements and performing some action on the element(s). Basic syntax is: $(selector).action() * A $ sign to define/access jQuery * A (selector) to "query (or find)" HTML elements * A jQuery action() to be performed on the element(s) Examples: $(this).hide() - hides the current element. $("p").hide() - hides all

elements. $(".test").hide() - hides all elements with class="test". $("#test").hide() - hides the element with id="test". jQuery uses CSS syntax to select elements.
